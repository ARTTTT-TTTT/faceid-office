# ✨ Quickstart ✨

## 🛠️ Project setup 🛠️

```bash
pnpm install
```

---

## 🚀 Compile and run 🚀

### 🧪 development

```bash
pnpm dev
```

### 🚀 production

```bash
pnpm build
```

```bash
pnpm start
```

### 🔎 watch

```bash
pnpm start:watch
```

---

## 🧹 Format documents 🧹

```bash
pnpm lint:fix
```
