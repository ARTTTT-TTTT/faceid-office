# ✨ Quickstart ✨

## 🛠️ Project setup 🛠️

```bash
pnpm install
```

## 🐳 Docker Compose Up 🐳

```bash
docker compose up -d
```

### ⚙️ Prisma Generate ⚙️

```bash
pnpm prisma generate
```

### 💾 Prisma Migrate Dev (Initial Migration) 💾

```bash
pnpm prisma migrate dev --name "init"
```

## 🚀 Compile and run 🚀

### 🧪 development

```bash
pnpm start
```

### 🔎 debug

```bash
pnpm start:debug
```

### 🚀 production

```bash
pnpm start:prod
```

## 🧹 Clean project 🧹

```bash
pnpm lint:fix
```

## 🆑 Delete node_modules 🆑

```bash
rm -rf node_modules pnpm-lock.yaml
```
